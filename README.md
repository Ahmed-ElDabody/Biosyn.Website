# Biosyn Pharmaceuticals — Website

Biosyn Pharmaceuticals is a fast-growing Egyptian pharmaceutical company committed to delivering premium-quality, accessible medications. We specialize in Pediatrics and Otolaryngology, blending scientific rigor, ethical sourcing, and human-centered design.

**A Commitment Towards Better Health.**

## Site structure

- `index.html` — Home
- `about.html` — About, Vision & Mission, Values, Therapeutic Areas, Research, Quality
- `products.html` — Marketed products
- `pipeline.html` — Development pipeline
- `resources.html` — Treatment articles, guidelines, case studies
- `contact.html` — Contact form & details
- `styles.css` / `page.css` — Stylesheets
- `script.js` — Shared JavaScript
- `assets/` — Images and logos

## Tech

Static HTML/CSS/JS. No build step required. Deployed via Netlify with automatic deploys from this repository.
